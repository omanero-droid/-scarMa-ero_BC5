# Spotify Analytics Assistant

Aplicación desarrollada con Streamlit que permite analizar el historial de Spotify mediante preguntas en lenguaje natural.

## Funcionalidades
- Análisis de artistas más escuchados
- Evolución temporal
- Patrones de escucha
- Generación automática de gráficos

## Tecnologías
- Python
- Streamlit
- OpenAI API
- Plotly
